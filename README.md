# Customer Behavior Dashboard

## 📊 Project Overview

The Customer Behavior Dashboard is a Data Analytics project designed to analyze customer purchasing patterns, subscription behavior, product category performance, and demographic trends.

The project combines Python, SQL, and Power BI to transform raw customer shopping data into meaningful business insights through interactive visualizations and KPI-driven reporting.


## 🎯 Objective

To understand customer behavior by analyzing:

- Customer spending patterns
- Product category performance
- Subscription trends
- Customer demographics
- Shipping preferences
- Review ratings

and present the findings in an interactive Power BI dashboard.


## 📂 Dataset

The dataset contains customer shopping information, including:

- Customer ID
- Age
- Gender
- Product Category
- Item Purchased
- Purchase Amount
- Review Rating
- Subscription Status
- Shipping Type
- Previous Purchases
- Promo Code Usage
- Payment Method

## 🔄 Project Workflow

### 1. Data Loading
- Imported dataset into Python.
- Loaded data into a Pandas DataFrame.

### 2. Data Cleaning
- Checked for missing values.
- Removed duplicate records.
- Verified data types.
- Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)
- Customer demographics analysis.
- Spending behavior analysis.
- Category-wise sales analysis.
- Subscription trend analysis.
- Rating distribution analysis.

### 4. SQL Analysis
Performed SQL queries to answer business questions such as:

- Revenue by Gender
- Revenue by Category
- Customer Segmentation
- Subscription Analysis
- Age Group Analysis
- Product Performance Analysis
- Shipping Preference Analysis

### 5. Dashboard Development
Created an interactive Power BI dashboard containing:

- KPI Cards
- Revenue Analysis
- Category Performance
- Customer Segmentation
- Subscription Insights
- Age Group Analysis
- Interactive Filters

### 6. Reporting
Generated business insights and recommendations based on dashboard findings.

## 🔍 Key Insights

- Non-subscribed customers account for the majority of customers.
- Clothing category generates the highest revenue.
- Clothing category records the highest number of sales.
- Senior customers contribute significantly to overall revenue.
- Customer review ratings remain consistently positive across categories.
